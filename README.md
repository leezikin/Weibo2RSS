# Weibo2RSS
base on [Weibo2RSS](http://github.com/DIYgod/Weibo2RSS)

> Weibo2RSS 改版

## Install

软件环境：Nodejs

## 介绍

RSS 格式输出一个微博博主最新的微博 和所有图片。配合 [IFTTT](https://ifttt.com/) 还可以实现更多好玩的功能。

##DEMO

http://weibo2rss-rss-builder.a3c1.starter-us-west-1.openshiftapps.com/

## 搭建

需要环境：Node.js
